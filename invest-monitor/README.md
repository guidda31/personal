# 투자 모니터링 봇 (KOSPI/KOSDAQ + USD/KRW)

관심 종목과 원/달러 환율을 추적해
- 현재가
- 목표가/상승여력
- 손절가/하락여력
- 시나리오 코멘트
를 한 번에 리포트합니다.

## 1) 설정
`config.json`의 종목/목표가/손절가를 수정하세요.

## 2) 실행
```bash
python3 invest-monitor/report.py
```

## 3) 출력 예시
- 기준일시
- USD/KRW
- 종목별: 현재가, 목표가(상승률), 손절가(하락률), 시나리오

## 4) 참고
- 국내 종목은 네이버 금융 시세 페이지를 사용합니다.
- 환율은 네이버 금융 marketindex를 사용합니다.
- 장외 시간/주말에는 마지막 반영값 기준으로 계산됩니다.
